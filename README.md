# 🔍 Multimodal Deception Suite Core Architecture

An artificial intelligence deception detection engine combining automated computer vision biometric tracking and vocal semantic parsing into a unified pipeline.

## 🚀 Features

* **Computer Vision Processing Layer:** Tracks double Eye Aspect Ratio (EAR) profiles using regional landmarks mapping through MediaPipe Face Mesh mesh configurations.
* **Audio Analytics Layer:** Performs real-time speech-to-text processing using ambient background noise normalization filtering.
* **Consolidated Truth Decision Matrix:** Correlates physiological stress behaviors (e.g., erratic blinking or freezing states) against verbal evasion keywords.
* **Modern Cyberpunk Theme UI:** Engineered with dark design principles, a responsive split monitoring grid, and animated tracking rails.

## 🛠️ Installation

### 1. Prerequisite Audio Bindings
Ensure your machine has system-level `portaudio` development headers installed:

* **Ubuntu/Debian:** `sudo apt-get install portaudio19-dev python3-pyaudio`
* **macOS:** `brew install portaudio`
